## Detailed Summary

This post explains Karpathy's follow-up to the original LLM wiki post (which got 10M+ views). The key insight:

**Share "idea files" instead of code.**

An idea file is:
- A text description of what you want to build
- The concept/schema/operating manual
- The "why" and "how" in structured form
- More portable than code; can guide multiple implementations

This directly supports treating files like `CLAUDE.md` as:
- First-class intellectual property
- Reusable across projects
- Worth sharing and maintaining separately from code
