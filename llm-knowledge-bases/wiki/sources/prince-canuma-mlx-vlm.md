## Detailed Summary

This post announces the MLX-VLM (multimodal vision language model) v0.4.3 release with significant capability expansions:

### New Model Support
- **Gemma 4** — multimodal with audio support
- **Falcon-OCR** — optical character recognition
- **Falcon Perception** — visual understanding
- **Granite Vision 4.0** — vision model

### New Capabilities
- **SAM 3.1** — segmentation anything with object multiplex
- **RF-DETR** — real-time detection and segmentation
