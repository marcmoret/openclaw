## Detailed Summary

This post captures the core operational pattern in accessible terms:

1. **Collection phase** — dump all raw sources (articles, papers, repos, datasets, images) into a single folder
2. **Organization phase** — point Claude at the folder to read and organize
3. **Token allocation shift** — more compute goes to knowledge management than coding

The insight that token spend is shifting suggests this is not just a nice-to-have pattern, but an economic reality: at scale, maintaining knowledge is as or more expensive than writing code.
