## Executive Summary
A comprehensive collection of 20 reusable agentic skills designed to work across Claude, ChatGPT, and Gemini platforms. Each skill is written in markdown format with name, description, overview, features, output format, instructions, and constraints. Skills are grouped into five categories: Writing & Content (5), Visual & Infographic (6), Research & Analysis (5), and Coding & Automation (4). This is a toolkit for enhancing any LLM agent with specialized workflows.

## Skill Categories and Overview

### Writing & Content Skills (5)
1. **SCQA Writing Framework** — Situation-Complication-Question-Answer structure for logical, engaging narratives
   - Use: Threads, articles, reports, educational material, storytelling
   - Key output: SCQA blocks with short paragraphs and bullet highlights

2. **Content Repurposing Engine** — Transform long-form content into multiple platform formats
   - Use: Convert blogs/notes/articles to social media, short videos, summaries
   - Key output: Platform-specific content with engaging hooks

3. **Tone & Style Enforcer** — Maintain consistent brand voice and style across outputs
   - Use: Ensure all generated content matches defined tone and guidelines
   - Key output: Text aligned with style guide, optional bullet structure

4. **Long-Form to Summary Compressor** — Condense complex content into digestible summaries
   - Use: Reduce long documents while preserving essential ideas
   - Key output: Concise paragraph or bullet points

5. **Structured Copywriting Skill** — Generate persuasive, well-structured marketing copy
   - Use: Marketing, articles, social media content with strong hooks
   - Key output: Sections with bullet points, hooks, CTA, conclusion

### Visual & Infographic Skills (6)
6. **Excalidraw Diagram Generator** — Convert textual concepts into diagram instructions
   - Use: Visualization, learning, planning, mapping workflows
   - Key output: Diagram title, nodes, connections, layout suggestion

7. **Infographic Builder** — Transform text into infographic-friendly format
   - Use: Reports, presentations, educational materials
   - Key output: Steps, headings, visual cues, optional icons

8. **Flowchart Decision Builder** — Generate decision trees and flowcharts from text
   - Use: Process simplification, decision-making workflows
   - Key output: Nodes, connections, layout guidance

9. **UI/UX Layout Advisor** — Optimize interface layouts for clarity and usability
   - Use: Design review, element positioning, hierarchy
   - Key output: Layout instructions, element positioning, visual hints

10. **Video Script Generator** — Produce structured scripts with hooks and pacing
    - Use: Short and long-form video content
    - Key output: Hook, content sections, clear CTA

11. **Video Editing Planner** — Suggest editing structure, scene cuts, transitions
    - Use: Improve video quality and engagement
    - Key output: Editing steps, scene notes, transition plan

### Research & Analysis Skills (5)
12. **Deep Research Synthesizer** — Synthesize large datasets into actionable insights
    - Use: Convert large text into structured insights and summaries
    - Key output: Key insights, supporting details, summary paragraph

13. **Onchain Transaction Analyzer** — Analyze blockchain transactions and token flows
    - Use: Decode crypto/blockchain data for non-experts
    - Key output: Step-by-step explanation, key actors, summary insights

14. **Source Validation Skill** — Evaluate credibility and biases in information sources
    - Use: Filter trustworthy content from research
    - Key output: Verified sources, reliability notes, key insights

15. **Competitive Intelligence Skill** — Compare products/tools/protocols with SWOT analysis
    - Use: Business, tech, market research
    - Key output: Bullet comparison, strengths/weaknesses, key takeaways

16. **Knowledge Structuring Skill** — Organize unstructured info into clear frameworks
    - Use: Transform messy input into usable knowledge
    - Key output: Structured framework, key points, optional notes

### Coding & Automation Skills (4)
17. **Code Review Skill** — Analyze code for bugs, inefficiencies, best practices
    - Use: Quality assurance, optimization, style enforcement
    - Key output: Issues found, suggested fixes, optional summary

18. **Workflow Automation Agent** — Break complex tasks into step-by-step workflows
    - Use: Goal → actionable workflows for AI or human execution
    - Key output: Goal, stepwise actions, tools & instructions

19. **Skill Creator (Meta Skill)** — Generate new AI skills in .md format
    - Use: Automate creation of reusable skills
    - Key output: Structured .md file with name, description, instructions

20. **DevOps Assistant** — Support version control, deployment, automation tasks
    - Use: Development workflows, CI/CD, versioning
    - Key output: Task instructions, stepwise guide, automation recommendations

## Key Patterns Across All Skills
1. **Structured input → structured output** — Every skill transforms unstructured or semi-structured input
2. **Markdown-based format** — Portable, version-control friendly, works across platforms
3. **Reusability** — Each skill can be copied directly into ChatGPT/Gemini/Claude
4. **Constraints listed** — Every skill includes guard rails (e.g., "Avoid generic summaries," "Maintain clarity")
5. **Cross-platform compatibility** — Same .md works in Claude, ChatGPT, Gemini (language model agnostic)
