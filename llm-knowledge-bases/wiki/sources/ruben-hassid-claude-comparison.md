## Detailed Summary

This post addresses a common problem: people default to one Claude tool when multiple tools are optimized for different tasks.

### The Tools
1. **Chat** — default, general-purpose conversational AI
2. **Code** — for coding tasks, project development
3. **Cowork** — for collaborative work and teamwork
4. **Projects** — for organizing and managing projects

### The Guidance
- Use **Code** when you're building/coding
- Use **Cowork** when you're working (collaboratively)
- Use **Projects** when you're organizing/managing
- Stop defaulting everything to Chat

## Why It Matters
