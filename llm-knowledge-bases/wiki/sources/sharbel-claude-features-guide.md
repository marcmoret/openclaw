## Executive Summary
A ranked guide to 20 Claude features organized by impact and utility. Most users (estimated 90% of paid subscribers) use only 10% of Claude's capabilities, primarily treating it as an advanced search engine. This guide highlights underutilized features that can transform how users work, from foundational (Projects, Custom Instructions) to advanced (Extended Thinking, Artifacts, GitHub Integration).

## Key Takeaways
- **Projects** are the single most underused feature but create persistent workspace context
- **Custom Instructions per Project** eliminate repetitive explanations and create consistent voice
- **Extended Thinking** makes Claude reason through complex problems (strategy, debugging, legal analysis)
- **Code Execution** runs Python, analyzes CSV data, creates visualizations—all inline
- **File Uploads** (PDFs, spreadsheets, images, code) reduce copy-paste friction
- **Vision** supports screenshots, diagrams, mockups, whiteboards, and charts
- **Artifacts** create shareable interactive tools, calculators, and charts live in-chat
- **GitHub Integration** connects to actual codebases, enabling PR review and bug analysis with real context
- **Voice Mode** enables conversational interaction on mobile and during brainstorming
- **Data Analysis Mode** + **File Creation** together form a data workflow (upload, analyze, export)
- **Claude Code** (terminal agent) is included in Max subscription but many Max users don't know it exists
- **Priority Access to New Features** means Max users get new models/capabilities days before others
- **Extended context windows** let users work with much larger files and codebases than Chat
- **Multiple models in one subscription** (Haiku, Sonnet, Opus) allow cost/quality optimization per task

## Detailed Feature Breakdown

### Basics Most People Skip
1. **Projects** — Persistent workspaces where all conversations share context, files, and instructions
2. **Custom Instructions per Project** — System prompt runs automatically on every conversation
3. **Project Files** — Upload documents once; Claude reads them at the start of every session

### Power Features Nobody Talks About
4. **Extended Thinking** — Toggle that makes Claude slow down and reason through complex problems
5. **Code Execution** — Write Python, execute it, see results/charts/tables inline
6. **File Uploads** — PDF, spreadsheet, image, code, Word, PowerPoint—Claude reads all
7. **Vision** — Paste images directly; Claude analyzes screenshots, mockups, diagrams
8. **Web Search** — Access internet in real time; enabled via toolbar icon

### Ones That Make You Noticeably Faster
9. **Artifacts** — Live preview panel for charts, tools, calculators, mini-apps; shareable links
10. **Data Analysis Mode** — Upload spreadsheet, ask Claude to find patterns, flag anomalies, visualize
11. **File Creation** — Generate downloadable PDFs, spreadsheets, slides directly from chat
12. **Voice Mode** — Full conversational back-and-forth (not just dictation); mobile and brainstorming

### Integrations Nobody Sets Up
13. **GitHub Integration** — Connect repo; Claude reads actual codebase, not snippets
14. **Google Drive** — Pull documents directly into context without copy-paste
15. **Zapier and Tool Integrations** — JIRA, Intercom, and growing list; native connections in settings

### Max-Specific Things
16. **Claude Code** — Terminal agent that reads codebase, plans tasks, ships changes (included in Max)
17. **Multiple Models in One Subscription** — Access Haiku (fast/cheap), Sonnet (general), Opus (reasoning)
18. **Priority Access to New Features** — Max users get new models/capabilities before Pro/free

### Habits Most People Never Build
19. **Start a New Chat Between Topics** — Context is finite; quality drops as conversations fill; new chat = fresh start
20. **Share Artifacts as Links** — Every artifact gets a public link; share the live tool instead of a screenshot
