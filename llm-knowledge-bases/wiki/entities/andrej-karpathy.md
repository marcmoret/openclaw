# Andrej Karpathy

## Summary

Andrej Karpathy is the central originating source for the core project pattern: a persistent LLM-maintained markdown wiki sitting between the human and raw source materials.

## Why relevant

His X post and `llm-wiki` gist are the strongest foundational sources for the current project. They define the main architecture and workflow ideas:
- raw sources
- compiled wiki
- schema/instruction layer
- ingest / query / lint
- compounding durable outputs
